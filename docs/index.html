Build from source

macOS 15+, Swift 6. No Xcode project — it's a SwiftPM package.

git clone https://github.com/SohrabZ/swiftui-macos-app.git cd swiftui-macos-app swift run

1 · Rename

The package in Package.swift, plus app.yml, AppInfo, and AppLog.

2 · Retheme

Edit the two palette tables — they share ThemeID and stay in sync.

3 · Retoken

Sizes, radii, fonts, and keys live in DesignSystem.swift.

4 · Extend

Add a settings section, a deep link, a string — the patterns are all there.
